# Graphscii

An application that prints ASCII representations of graphs! Here is the desired
output for the Kuratowski Graph K3,3:

```
   ┌─┐
 ┌─┤ ├────┐
 │ └┬┘    │
┌┴┐ │ ┌─┐┌┴┐
│ ├─│─┤ ├┤ │
└┬┘ │ └┬┘└┬┘
 │  │ ┌┴┐┌┴┐
 │  └─┤ ├┤ │
 │    └─┘└┬┘
 └────────┘
```

This program is currently a work in progress - check back later!
